# Калькулятор вкладов

Выполнил: Волков Даниил
Консольное приложение: Ковшов Андрей

Раскладка по директориям:

 - 'application' содержит main файл для консольного приложения
  - `include` содержит публичные заголовочные файлы, предназначенные для
    включения в пользовательские проекты.
  - `src` содержит реализацию класса Digits.
  - `test` содержит модульные тесты для класса.
